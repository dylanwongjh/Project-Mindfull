PROJECT MINDFULL

Project Mindfull is a simple AI-powered chatbot designed to provide a friendly and supportive space for discussions about mental well-being. It's built to be a safe, accessible tool for anyone looking for an empathetic listening ear and helpful resources.

Features
Empathetic Conversations: Engage in natural, supportive conversations powered by Google's Gemini AI.

Helpful Resources: Access a list of local and international mental health resources for immediate support.

Intuitive UI: A clean and minimalist user interface with chat bubbles that mimic a familiar messaging app.
